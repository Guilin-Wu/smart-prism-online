# 测试指南

## 🚀 快速测试方法

### 方法1：直接打开（最简单）

1. **打开文件**
   - 在文件管理器中找到 `index.html`
   - 双击打开，或右键选择"用浏览器打开"

2. **推荐浏览器**
   - Chrome（推荐）
   - Edge
   - Firefox
   - Safari

### 方法2：使用本地服务器（推荐）

某些功能（如ES6模块、AJAX）需要服务器环境才能正常工作。

#### 使用 Python（如果已安装）

```bash
# Python 3
cd /Users/glwu/Smart-Prism_2
python3 -m http.server 8000

# 然后在浏览器访问: http://localhost:8000
```

#### 使用 Node.js（如果已安装）

```bash
# 安装 http-server（如果还没安装）
npm install -g http-server

# 启动服务器
cd /Users/glwu/Smart-Prism_2
http-server -p 8000

# 然后在浏览器访问: http://localhost:8000
```

#### 使用 VS Code Live Server

1. 安装 VS Code 扩展 "Live Server"
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

## 🔍 常见问题排查

### 问题1：样式没有加载

**症状**：页面显示没有样式，只有纯文本

**解决方法**：
1. 检查浏览器控制台（F12）是否有404错误
2. 确认 `css/main.css` 文件存在
3. 检查文件路径是否正确

### 问题2：Logo 不显示

**症状**：Logo 位置显示为空白或破损图标

**解决方法**：
1. 检查 `assets/images/logo.png` 是否存在
2. 检查 `index.html` 中的路径是否为 `./assets/images/logo.png`

### 问题3：某些功能不工作

**症状**：点击按钮没有反应

**可能原因**：
- JavaScript 文件（`script.js`）还未迁移，功能应该正常
- 如果使用 ES6 模块，需要服务器环境

### 问题4：控制台有错误

**检查步骤**：
1. 按 F12 打开开发者工具
2. 查看 Console 标签页
3. 记录错误信息
4. 检查 Network 标签页，看是否有文件加载失败
